## 주요 기능

#### ⌚️ 워치 진동 메트로놈
- 음악 BPM에 따른 애플워치 진동 메트로놈
- 워치 음악 컨트롤 기능 (재생/일시정지)

#### 🎶 소리로 멜로디, 메트로놈 동시 재생
- 음악 멜로디와 메트로놈이 소리로 동시 재생

#### 🎼 내 악보를 소리,진동으로 변환
- PDF 악보를 사운드,햅틱 시퀀스로 변경하여 저장

#### 🛜 서버 구축
- User별 PDF → 음악 변환 상태값 확인 가능
- 변환 완료시 알림 및 저장
